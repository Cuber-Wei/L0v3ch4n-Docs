Schmidt-Samoa密码系统

约 335 字大约 1 分钟

CTFCryptoSchmidt-Samoa

2025-03-12

Schmidt-Samoa 密码系统，像 Rabin 加密一样，其安全性基于整数因式分解的难度。但 Rabin 解密时会得到四个解，而 Schmidt-Samoa 得到的是唯一解。

加密过程

密钥生成

  1. 选取两个大的质数 p 和 q 并进行计算 N=p2∗qN = p^2*qN=p2∗q
  2. 计算 d=invert(N,φ(pq))d=invert(N,\varphi(pq))d=invert(N,φ(pq))
  3. 公钥取数对(N,d)(N, d)(N,d)

加密

对消息 mmm ，计算密文 C=mN(modN)C=m^N \pmod NC=mN(modN)

解密

计算明文 m=Cd(modpq)m = C^d\pmod{pq}m=Cd(modpq)

正确性证明就不写了。摸了摸了

关于 pq 的获取问题

我们有N=p2q,d⋅N≡1(mod(p−1)(q−1))N=p^2q,d\cdot N\equiv1\pmod{(p-1)(q-1)}N=p2q,d⋅N≡1(mod(p−1)(q−1))，有欧拉定理：

a(p−1)(q−1)≡1(modpq)a^{(p-1)(q-1)}\equiv1\pmod{pq} a(p−1)(q−1)≡1(modpq)

所以：

a(p−1)(q−1)⋅a≡1(modpq)a^{(p-1)(q-1)}\cdot a\equiv1\pmod{pq} a(p−1)(q−1)⋅a≡1(modpq)

a≡a(p−1)(q−1)(modpq)a\equiv a^{(p-1)(q-1)}\pmod{pq} a≡a(p−1)(q−1)(modpq)

由此，我们有：

k⋅pq=aNd−ak\cdot pq=a^{Nd}-a k⋅pq=aNd−a

pq=gcd(aNd−a,N)pq=gcd(a^{Nd}-a,N) pq=gcd(aNd−a,N)

我们只要随便取个aaa就行能拿到pqpqpq了。

例题

from Crypto.Util.number import *
flag = b'NSSCTF{******}'

p = getPrime(512)
q = getPrime(512)
n = p*p*q
e = n

c = pow(bytes_to_long(flag), e, n)
d = inverse(e, (p-1)*(q-1))

xnd≡xnd−1⋅x≡x(modpq)x^{nd}\equiv x^{nd-1}\cdot x\equiv x\pmod{pq} xnd≡xnd−1⋅x≡x(modpq)

显然有 pq∣(xnd−x)pq\mid{(x^{nd}-x)}pq∣(xnd−x) 。然后题目有n=p2qn=p^2qn=p2q ，GCD 一下就有pqpqpq了。
